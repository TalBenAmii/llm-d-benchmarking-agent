The llm-d Benchmarking Assistant ({{ include "agent.fullname" . }}) is deploying.

1. Watch it come up:
   kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "agent.fullname" . }}

2. Open the chat UI (port-forward):
   kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "agent.fullname" . }} 8000:{{ .Values.service.port }}
   # then browse http://127.0.0.1:8000

3. Check health / readiness:
   kubectl -n {{ .Release.Namespace }} get pods -l app.kubernetes.io/instance={{ .Release.Name }}
